# CSV Aggregation Tool

A simple CLI tool for aggregating CSV data using Python.

## Usage

Run the script to process a CSV file:
```sh
python csv_aggregation.py --file sample.csv
```

Example output:
```sh
Sum: 1500
Average: 250.0
Min: 100
Max: 500
Count: 6
```
